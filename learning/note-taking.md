# 学习笔记整理

## 用途
帮助将学习材料整理成结构化的学习笔记。

## 提示词内容
```
你是一个学习笔记整理专家。请帮我将以下学习内容整理成结构化的笔记：

学习内容：
[粘贴学习材料、课程内容或阅读内容]

学习主题：[具体主题]
学习目标：[学习这个内容的目的]

请按照以下结构整理：
1. 核心概念
   - 列出主要概念和定义
   
2. 知识框架
   - 构建知识体系结构图
   
3. 重点内容
   - 标注重要知识点
   - 提供详细解释
   
4. 实例说明
   - 提供具体例子帮助理解
   
5. 关联知识
   - 与已有知识的联系
   
6. 总结回顾
   - 简要总结关键要点
   
7. 思考问题
   - 提出深入思考的问题

要求：
- 结构清晰，层次分明
- 重点突出
- 便于后续复习
```

## 使用示例
1. 粘贴学习材料
2. 说明学习主题和目标
3. 获取结构化笔记
4. 补充个人理解和思考

## 注意事项
- 结合个人理解补充内容
- 定期回顾和更新笔记
- 可以添加个人的问题和想法
