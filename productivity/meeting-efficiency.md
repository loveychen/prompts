# 会议效率提升

## 用途
帮助提高会议效率，确保会议有成果、有产出。

## 提示词内容
```
你是一个会议效率专家。请帮我规划和优化会议：

会议信息：
- 会议主题：[具体议题]
- 会议目标：[期望达成的结果]
- 参会人员：[人员及角色]
- 会议时长：[预计时长]
- 会议类型：[头脑风暴/决策会议/同步会议等]

请提供：
1. 会前准备
   - 会议议程（时间分配）
   - 需要准备的材料
   - 参会人员准备事项
   - 会议规则
   
2. 会议流程
   - 开场介绍
   - 议题讨论顺序
   - 时间控制要点
   - 决策流程
   
3. 会议主持要点
   - 如何引导讨论
   - 如何控制时间
   - 如何达成共识
   
4. 会议记录模板
   - 需要记录的要点
   - 行动项格式
   
5. 会后跟进
   - 会议纪要要点
   - 行动项跟踪
   - 效果评估

要求：
- 目标明确
- 流程高效
- 确保产出
- 便于跟踪
```

## 使用示例
1. 确定会议主题和目标
2. 获取会议流程建议
3. 会前做好充分准备
4. 按流程主持会议
5. 会后及时跟进

## 注意事项
- 会前充分准备
- 严格控制时间
- 确保所有人参与
- 明确行动项和责任人
- 及时跟进会议决议
