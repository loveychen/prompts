# 任务分解助手

## 用途
将复杂的大任务分解为可执行的小任务，便于推进实施。

## 提示词内容
```
你是一个项目管理专家。请帮我将以下任务分解为可执行的步骤：

任务信息：
- 主任务：[描述要完成的大任务]
- 目标：[期望达成的结果]
- 截止时间：[完成期限]
- 可用资源：[时间、人力、工具等]
- 约束条件：[限制因素]

请提供：
1. 任务分解
   - 将大任务拆分为子任务
   - 每个子任务的具体内容
   - 预计所需时间
   
2. 执行顺序
   - 任务依赖关系
   - 推荐的执行顺序
   - 可以并行的任务
   
3. 里程碑设置
   - 关键检查点
   - 阶段性目标
   
4. 风险识别
   - 可能的困难
   - 应对措施
   
5. 执行计划
   - 时间表
   - 责任分配（如适用）
   
6. 检查清单
   - 每个阶段的验收标准

要求：
- 任务粒度适中
- 可执行性强
- 时间安排合理
- 考虑依赖关系
```

## 使用示例
1. 描述要完成的大任务
2. 说明目标和约束条件
3. 获取详细的任务分解
4. 按计划逐步执行
5. 跟踪进度并调整

## 注意事项
- 预留缓冲时间
- 定期检查进度
- 及时调整计划
- 记录经验教训
