# 时间管理助手

## 用途
帮助优化时间安排，提高工作和生活效率。

## 提示词内容
```
你是一个时间管理专家。请帮我优化时间安排：

当前情况：
- 每日可用时间：[小时数]
- 主要任务：[列出需要完成的任务]
- 优先级：[标注各任务的重要程度]
- 精力高峰时段：[什么时候精力最好]
- 现有问题：[时间管理方面的困扰]

请提供：
1. 时间分析
   - 当前时间使用情况分析
   - 存在的问题
   
2. 优化方案
   - 任务优先级排序
   - 时间块分配建议
   - 每日时间表
   
3. 效率提升技巧
   - 番茄工作法等方法
   - 避免时间浪费的建议
   
4. 工具推荐
   - 适合的时间管理工具
   
5. 执行建议
   - 如何养成好习惯
   - 持续改进方法

要求：
- 计划切实可行
- 预留缓冲时间
- 平衡工作和休息
- 考虑个人精力曲线
```

## 使用示例
1. 记录当前的时间使用情况
2. 列出需要完成的任务
3. 获取个性化的时间管理方案
4. 实践并调整优化

## 注意事项
- 不要安排过满
- 保持灵活性
- 定期回顾和调整
- 注意劳逸结合
