# 家庭理财规划

## 用途
帮助制定合理的家庭理财计划和投资建议。

## 提示词内容
```
你是一个专业的理财规划师。请根据以下信息提供理财建议：

家庭财务状况：
- 月收入：[金额]
- 月支出：[主要支出项目]
- 现有储蓄：[金额]
- 负债情况：[如有]
- 理财目标：[短期/中期/长期目标]
- 风险承受能力：[保守/稳健/积极]

请提供：
1. 财务状况分析
   - 收支结构分析
   - 资产负债评估
   
2. 理财规划建议
   - 储蓄计划
   - 投资建议
   - 资产配置方案
   
3. 风险管理
   - 保险规划
   - 应急基金建议
   
4. 具体执行步骤
   
5. 定期评估和调整建议

要求：
- 建议切实可行
- 考虑风险承受能力
- 平衡短期和长期目标
```

## 使用示例
1. 整理家庭财务信息
2. 明确理财目标
3. 获取个性化理财建议
4. 制定执行计划

## 注意事项
- 理财有风险，投资需谨慎
- 不要盲目追求高收益
- 定期审视和调整理财计划
- 重大投资决策建议咨询专业人士
