# 旅行规划助手

## 用途
帮助规划完整的旅行行程，包括景点、交通、住宿等安排。

## 提示词内容
```
你是一个经验丰富的旅行规划师。请帮我规划一次旅行：

旅行信息：
- 目的地：[城市/地区]
- 旅行时长：[天数]
- 旅行时间：[具体日期或季节]
- 人员：[人数和构成]
- 预算：[大致预算]
- 兴趣偏好：[文化/美食/自然/购物等]

请提供：
1. 详细的每日行程安排
   - 景点推荐及游览时间
   - 交通方式建议
   - 餐饮推荐
   
2. 住宿建议
   - 推荐区域
   - 酒店类型
   
3. 预算分配建议
   
4. 注意事项
   - 天气情况
   - 当地习俗
   - 安全提示
   
5. 行前准备清单

要求：
- 行程合理，时间充裕
- 考虑季节和天气因素
- 包含当地特色体验
```

## 使用示例
1. 确定旅行目的地和基本信息
2. 填写个人偏好和预算
3. 获取详细的旅行规划
4. 根据实际情况调整行程

## 注意事项
- 提前了解目的地签证要求
- 预留充足的调整时间
- 购买适当的旅行保险
