# 健康饮食规划

## 用途
根据个人需求制定健康的饮食计划和营养建议。

## 提示词内容
```
你是一个专业的营养师。请根据以下信息为我制定一份健康的饮食计划：

个人信息：
- 年龄：[年龄]
- 性别：[性别]
- 身高/体重：[身高/体重]
- 健康目标：[减重/增肌/维持健康等]
- 饮食偏好：[素食/无特殊要求等]
- 过敏信息：[如有]

请提供：
1. 每日营养需求分析
2. 一周饮食计划（三餐+加餐）
3. 推荐的食材清单
4. 简单的食谱建议
5. 注意事项和健康建议

要求：
- 营养均衡
- 易于执行
- 符合个人偏好和目标
```

## 使用示例
1. 填写个人健康信息
2. 明确饮食目标
3. 获取个性化的饮食计划
4. 根据实际情况调整执行

## 注意事项
- 如有特殊健康状况，请咨询专业医生
- 循序渐进，不要急于求成
- 定期评估和调整计划
