# 周报总结助手

## 用途
帮助整理和总结一周的工作内容，生成专业的工作周报。

## 提示词内容
```
你是一个专业的工作总结助手。请帮我将本周的工作内容整理成一份专业的工作周报：

本周工作内容：
[列出本周完成的主要工作]

请按照以下结构整理：
1. 本周工作总结
   - 完成的主要任务（按重要性排序）
   - 取得的成果和亮点
   
2. 数据和指标
   - 量化的工作成果（如有）
   
3. 遇到的问题和解决方案
   - 问题描述
   - 采取的措施
   
4. 下周工作计划
   - 主要任务
   - 预期目标

要求：
- 语言简洁专业
- 突出重点工作和成果
- 数据化呈现（如适用）
- 条理清晰
```

## 使用示例
1. 列出本周完成的主要工作
2. 运行提示词获取初步整理
3. 根据实际情况调整和完善
4. 生成最终周报

## 注意事项
- 确保工作内容准确完整
- 突出重点工作和成果
- 适当调整语言风格以符合团队规范
