# 技术文档撰写助手

## 用途
帮助撰写清晰、专业的技术文档，包括 API 文档、设计文档等。

## 提示词内容
```
你是一个专业的技术文档撰写专家。请帮我撰写一份技术文档：

文档类型：[API文档/设计文档/用户手册等]
主题：[具体主题]
受众：[目标读者群体]

请包含以下部分：
1. 概述：简要说明文档的目的和范围
2. 详细内容：根据文档类型提供相应的详细信息
3. 示例：提供具体的使用示例
4. 注意事项：列出需要特别注意的事项
5. 参考资料：相关的参考链接或资源

要求：
- 语言清晰简洁
- 结构层次分明
- 适当使用图表和示例
- 考虑读者的技术背景
```

## 使用示例
1. 确定文档类型和主题
2. 填写目标读者信息
3. 根据需要调整各个部分的要求
4. 获取专业的技术文档初稿

## 注意事项
- 根据实际项目调整文档结构
- 确保技术术语的准确性
- 可以分多次迭代完善文档
