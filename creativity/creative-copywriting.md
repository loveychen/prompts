# 创意文案撰写

## 用途
帮助创作吸引人的营销文案和创意内容。

## 提示词内容
```
你是一个创意文案专家。请为以下需求创作文案：

文案需求：
- 类型：[广告文案/社交媒体文案/产品介绍等]
- 主题：[具体主题]
- 目标受众：[受众群体特征]
- 传播渠道：[微信/微博/小红书等]
- 核心信息：[要传达的关键信息]
- 语言风格：[专业/活泼/幽默/文艺等]

请提供：
1. 多个文案版本（3-5个）
   - 不同的创意角度
   - 各有特色
   
2. 标题建议
   - 吸引眼球的标题
   
3. 配图建议
   - 适合的视觉元素
   
4. 发布建议
   - 最佳发布时间
   - 互动策略

要求：
- 符合目标受众特点
- 突出核心信息
- 具有吸引力和传播力
- 简洁有力
```

## 使用示例
1. 明确文案类型和目标
2. 提供必要的背景信息
3. 获取多个创意版本
4. 选择最适合的版本或综合改进

## 注意事项
- 确保内容真实准确
- 避免夸大宣传
- 注意版权和法律问题
- 根据反馈不断优化
