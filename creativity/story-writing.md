# 故事创作助手

## 用途
帮助创作引人入胜的故事，包括小说、剧本等创意写作。

## 提示词内容
```
你是一个经验丰富的故事创作者。请帮我创作一个故事：

故事设定：
- 类型：[科幻/奇幻/现实/悬疑/爱情等]
- 主题：[核心主题]
- 长度：[短篇/中篇/长篇]
- 风格：[轻松/严肃/幽默/悲伤等]
- 目标读者：[读者群体]

故事要素（可选）：
- 主要角色：[角色设定]
- 背景设定：[时间、地点、世界观]
- 核心冲突：[主要矛盾]
- 期望结局：[开放式/圆满/悲剧等]

请提供：
1. 故事大纲
   - 起承转合的完整结构
   - 主要情节点
   
2. 人物设定
   - 主要角色性格特点
   - 人物关系
   
3. 详细内容
   - 开头部分（吸引读者）
   - 主体情节展开
   - 高潮部分
   - 结局
   
4. 创作建议
   - 需要注意的细节
   - 可以深化的地方

要求：
- 情节引人入胜
- 人物性格鲜明
- 逻辑自洽
- 有情感共鸣
```

## 使用示例
1. 确定故事类型和主题
2. 提供基本的故事设定
3. 获取故事大纲和初稿
4. 根据反馈迭代完善
5. 润色和修改细节

## 注意事项
- 保持故事的连贯性
- 注意人物行为的合理性
- 避免情节老套
- 多次修改打磨
- 可以分章节逐步创作
