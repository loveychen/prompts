# 头脑风暴助手

## 用途
帮助进行创意思维，产生大量有价值的创意想法。

## 提示词内容
```
你是一个创意思维专家。请针对以下主题进行头脑风暴：

头脑风暴主题：[具体主题或问题]
背景信息：[相关背景]
目标：[希望达成的目标]
限制条件：[如有]

请使用以下思维方法产生创意：
1. 自由联想
   - 尽可能多的想法
   - 不限制可行性
   
2. 反向思考
   - 从相反角度思考
   
3. 组合创新
   - 结合不同领域的想法
   
4. 类比思维
   - 从其他领域借鉴
   
5. 场景想象
   - 想象不同的使用场景

请提供：
- 至少20个创意想法
- 每个想法的简要说明
- 标注特别有潜力的想法
- 对重点想法进行深入分析

要求：
- 数量优先，暂不考虑可行性
- 思路多样化
- 大胆创新
```

## 使用示例
1. 明确头脑风暴的主题
2. 提供必要的背景信息
3. 获取大量创意想法
4. 筛选和评估可行的创意
5. 深入发展选中的创意

## 注意事项
- 在头脑风暴阶段不要过早否定想法
- 鼓励跨界思考
- 记录所有想法
- 后期再进行可行性评估
