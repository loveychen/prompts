# 提示词模板 (Prompt Template)

使用此模板创建新的提示词文件。

## 中文版

```markdown
# 提示词标题

## 用途
描述这个提示词的用途和适用场景

## 提示词内容
\```
[在这里粘贴完整的提示词内容]
\```

## 使用示例
1. 第一步：...
2. 第二步：...
3. 第三步：...

## 注意事项
- 注意事项1
- 注意事项2
- 注意事项3
```

---

## English Version

```markdown
# Prompt Title

## Purpose
Describe the purpose and applicable scenarios of this prompt

## Prompt Content
\```
[Paste the complete prompt content here]
\```

1. Step 1: ...
2. Step 2: ...
3. Step 3: ...

- Note 1
- Note 2
- Note 3
```

---

## 使用说明 (Instructions)

1. 复制上面的模板
2. 根据实际情况填写各个部分
3. 在对应的分类目录下创建新文件
4. 文件命名使用小写字母和连字符，如：`my-prompt-name.md`

## 建议 (Suggestions)

- **标题**：简洁明了，能准确反映提示词的用途
- **用途**：说明什么时候使用这个提示词，能解决什么问题
- **内容**：完整的提示词，可以直接复制使用
- **示例**：提供具体的使用步骤，帮助理解
- **注意事项**：列出重要的提醒和限制条件
