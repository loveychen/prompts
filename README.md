# prompts

个人 prompt 仓库，用于分门别类保存当前工作和生活中的高效率 prompts。

## 目录结构

```
academics/
	paper-reading/
	PICOT 论文结构化分析.md
english/
work-skills/
life/
```

## 目录说明

- academics：论文阅读、学术写作与研究方法相关
- academics/paper-reading：论文阅读与结构化分析的 prompts
- english：英语学习、写作与口语练习
- work-skills：工作技能、工具使用与流程优化
- life：生活效率、习惯、健康等

## 使用约定

- 每个 prompt 单独一个 `.md` 文件，文件名尽量能直接反映用途
- 在文件开头写清楚适用场景、输入输出格式和示例
- 需要长期使用的 prompt 可以加上版本号或日期
